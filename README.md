# Amazon Clone

### Description: 
This project is an Amazon clone created using HTML and CSS to replicate the design and functionality of the Amazon e-commerce website.

![Screenshot 2024-01-04 125346](https://github.com/aakasshhh/Amazon-clone/assets/118706951/b806ec6c-a510-473f-a61b-e6eae3b07ffd)

### Features:
- Search functionality
- Product listings
- User authentication

### Installation Instructions:
To run this project locally, clone the repository and open the index.html file in a web browser.


### Usage: 
Explore the various features of the Amazon clone, including browsing products and simulating a user login. 


![Screenshot 2024-01-04 125428](https://github.com/aakasshhh/Amazon-clone/assets/118706951/f402875f-ada8-486c-b1f6-7ac59f3c232a)
